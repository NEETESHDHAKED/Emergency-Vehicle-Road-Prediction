# Emergency Vehicle Road Prediction

## Overview

Emergency Vehicle Road Prediction is an End-to-End Machine Learning project designed to predict road suitability and routing priority for emergency vehicles based on traffic and road conditions. The system helps identify the most suitable route for emergency response by analyzing multiple road and traffic-related features.

## Features

- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Feature Normalization
- Machine Learning Model Training
- Model Evaluation
- Real-Time Prediction
- Interactive Flask Web Dashboard
